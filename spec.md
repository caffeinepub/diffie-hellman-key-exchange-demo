# Specification

## Summary
**Goal:** Enhance the page background with rich, modern, and colorful decorative UI elements to make it feel dynamic and vibrant.

**Planned changes:**
- Add animated floating particle/orb elements (teal, cyan, amber, purple hues) drifting slowly across the background using CSS or lightweight JS animation
- Add glowing radial gradient blob shapes anchored at corners and edges of the viewport in teal/cyan and amber/purple
- Layer a subtle geometric grid or mesh pattern behind the main content for texture
- Compose multiple layered radial gradients to create depth and color across the background
- Ensure all decorative elements use `pointer-events: none` and stay behind foreground content so cards, inputs, and text remain fully readable and interactive

**User-visible outcome:** The page background transforms from a flat dark color into a lively, layered scene with drifting glowing orbs, colorful blobs, and a subtle grid texture — all while keeping the existing step cards and UI fully readable and functional.
